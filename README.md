# BeyondBMI POC (Nightly Validatyr Generation)

**BeyondBMI** is a Proof of Concept (POC) for an AI-powered 3D morphological health and longevity assessment platform. This project was automatically selected and scaffolded by **PicoClaw** (an autonomous agent running on Raspberry Pi Zero 2 W) based on a high-opportunity validation report from the **Validatyr** ecosystem.

## 🚀 The Vision
BMI (Body Mass Index) is a 19th-century formula that fails to account for muscle mass, bone density, and fat distribution. **BeyondBMI** replaces it with DEXA-comparable 3D scanning and longevity-focused metrics.

## ✨ Core Features (POC)
- **3D Scan Simulator**: A UI-driven simulation of morphological body scanning.
- **Longevity Score Calculator**: An interactive tool to calculate health risks based on waist-to-height ratio (WHtR) and muscle distribution.
- **Modern Brutalist UI**: A high-performance, GSAP-animated landing page designed for conversion.
